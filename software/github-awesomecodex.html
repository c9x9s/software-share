项目简介

A curated list of practical Codex skills for automating workflows across the Codex CLI and API.

awesome-codex-skills 是由 ComposioHQ 维护的 Codex 技能精选列表。Codex 是 OpenAI 推出的 AI 编程工具，这个项目收集了大量实用的工作流自动化技能。

什么是 Codex Skills

Codex Skills 是预定义的提示词模板，让 Codex 能够执行特定任务。比如自动生成文档、代码审查、Bug 修复等。

核心内容

🎯 精选技能

  • 代码审查 — 自动化代码质量检查
  • 文档生成 — 自动生成 API 文档
  • 测试生成 — 自动生成单元测试
  • 代码重构 — 批量代码优化
  • Bug 修复 — 智能问题定位

✅ 优点

  • 精选列表，质量有保证
  • 开箱即用
  • 持续更新
  • 社区驱动

❌ 缺点

  • 需要 Codex 订阅
  • 部分技能偏特定场景

总结

适合使用 Codex，这个列表能帮你快速找到实用的技能。4.9k Stars 说明它确实有用。
