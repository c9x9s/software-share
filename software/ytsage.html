YTSage

🖥️ Win / Mac / Linux | 💰 免费开源 | 📅 2024年更新
YouTube 视频下载 GUI 开源

简洁好用的 YouTube 下载器

yt-dlp 是目前最强的 YouTube 下载命令行工具，但命令行对普通用户不够友好。YTSage 做了个图形界面，让下载 YouTube 视频变得像点按钮一样简单。

基于 PySide6 开发，界面现代化，操作直观。支持视频、音频、字幕下载，还整合了 SponsorBlock 功能，可以自动跳过视频中的广告片段。

核心功能

⚡ YTSage 能做什么

  • 视频下载 — 支持任意画质，从 360P 到 4K
  • 音频提取 — 下载纯音频，支持 MP3、AAC 等格式
  • 字幕下载 — 自动下载字幕，可选择语言
  • SponsorBlock — 自动跳过片头广告、赞助商片段
  • 播放列表 — 支持整个播放列表批量下载
  • 简洁界面 — 现代化的图形界面，不用记命令

使用体验

只需要粘贴 YouTube 链接，选择想要的画质和格式，点击下载就完成了。相比直接用 yt-dlp 命令行，省去了记参数的时间。

SponsorBlock 功能是亮点。看教程视频时再也不用手动快进跳过赞助商内容了，下载的时候就自动处理好了，非常贴心。

✅ 优点

  • 图形界面，操作简单
  • 基于 yt-dlp，支持全平台
  • SponsorBlock 功能实用
  • 支持播放列表批量下载
  • 开源免费
  • 持续更新

❌ 缺点

  • 需要先安装 yt-dlp
  • 不支持哔哩哔哩等其他平台
  • Mac/Linux 安装稍微麻烦

适合人群

适合需要下载 YouTube 视频的用户，特别是喜欢看教程、纪录片、技术视频的朋友。纯命令行恐惧者用这个版本更舒服。

总结

YTSage 是 yt-dlp 的最佳图形界面伴侣。让复杂的命令行工具变得人人可用，值得一试。
