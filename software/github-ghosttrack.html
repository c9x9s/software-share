GhostTrack

⭐ 11,718 Stars | 📅 今日 1,033 Stars
Python OSINT 位置追踪 🔥 今日趋势

项目简介

Useful tool to track location or mobile number — 通过手机号进行定位追踪的开源工具。

GhostTrack 支持通过手机号在全球 3000+ 站点收集目标的信息。对于安全研究人员和 OSINT 调查者来说，这是一个有力的工具。

⚠️ 免责声明

🔒 法律提示

此工具仅供合法的安全研究和授权的渗透测试使用。未经授权对他人进行追踪在大多数国家/地区是违法行为。

技术原理

GhostTrack 通过自动化方式在各大社交媒体和网站查询手机号，利用这些平台的 API 或网页结构获取信息汇总。

✅ 优点

  • 自动化程度高
  • 支持站点多
  • 持续更新
  • 适合安全研究

⚠️ 注意事项

  • 仅供授权使用
  • 需要技术基础
  • 部分站点可能限制

总结

GhostTrack 是 OSINT 领域的热门工具。对于安全研究人员和渗透测试者来说，它能大幅提升信息收集效率。
