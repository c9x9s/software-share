uBlock Origin

⭐ 64,093 Stars | 🏆 All-Time Popular
浏览器扩展 广告拦截 🏆 长期高赞

项目简介

uBlock Origin 是 An efficient blocker for Chromium and Firefox. Fast and lean. 由 Raymond Hill 开发，是目前最高效的广告拦截器之一。

与其他广告拦截器相比，uBlock Origin 内存占用极低，同时拦截能力毫不逊色。

为什么选择 uBlock Origin

⚡ 核心优势

  • 极低资源占用 — CPU 和内存占用远低于 AdBlock Plus
  • 强大的拦截能力 — 默认规则库 + 自定义规则
  • 开源透明 — 代码完全开放，可审计
  • 隐私保护 — 默认屏蔽跟踪器

使用技巧

默认配置已经很好，但如果想更极致：

  • 启用「糟糕的域名」列表 — 屏蔽更多跟踪器
  • 使用 EasyList + EasyPrivacy — 中文网站最好加上 EasyChina
  • 临时允许 — 按住 Ctrl 点击，临时放行某个域名

✅ 优点

  • 资源占用极低
  • 拦截效率高
  • 完全免费开源
  • 支持 Chromium 和 Firefox
  • 自定义规则灵活

❌ 缺点

  • 界面比较极客
  • 规则配置有学习成本
  • 可能影响某些网站功能

总结

uBlock Origin 是每个浏览器必备的扩展。轻量、高效、免费，让你的浏览体验干净清爽。
