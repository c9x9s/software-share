DeepSeek to API

⭐ 2,796 Stars | 🏆 All-Time Popular
API DeepSeek 代理 🏆 长期高赞

项目简介

LLM API 聚合器 — DeepSeek/Claude/GPT 统一接口中间件。

这个项目让你可以通过统一的 API 接口访问多种 LLM 服务，支持 DeepSeek、Claude、GPT 等。它是一个高性能的 Go 语言中间件。

核心功能

🛠️ 技术特性

  • 统一接口 — 一个 API key，访问所有模型
  • 多账号轮询 — 自动轮换多个 API key
  • 格式兼容 — 兼容 OpenAI/Claude API 格式
  • Docker 部署 — 一键部署，开箱即用

使用场景

适合需要使用多种 LLM 但不想管理多个 API key 的开发者：

  • 统一管理多个 LLM 服务
  • 负载均衡，降低单点失败
  • 成本优化，自动选择性价比最高的模型

✅ 优点

  • Go 编写，性能优秀
  • 部署简单
  • 兼容性好
  • 支持 Docker

❌ 缺点

  • 需要自己提供 API key
  • 无官方维护
  • 生产环境需自行评估

总结

对于需要聚合多个 LLM 服务的开发者来说，这是一个实用的工具。2.7k Stars 说明它解决了真实的痛点。
