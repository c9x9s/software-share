项目简介

🕵️‍♂️ Collect a dossier on a person by username from 3000+ sites.

Maigret 是一个强大的 OSINT（开源情报）工具，通过用户名在 3000+ 网站搜索目标人物，收集他们的公开信息。

⚠️ 免责声明

🔒 法律提示

此工具仅供合法的安全研究和授权测试使用。未经授权收集他人信息在大多数国家/地区是违法行为。

核心功能

  • 多平台搜索 — 支持 3000+ 社交媒体和网站
  • 自动检测 — 智能识别目标在哪些平台有账号
  • 报告生成 — 生成详细的调查报告
  • API 支持 — 可以集成到其他工具中

✅ 优点

  • 覆盖平台极多
  • 自动化程度高
  • 开源可审计
  • 持续更新

⚠️ 注意

  • 仅供合法用途
  • 需要技术基础
  • 部分平台可能限制

总结

Maigret 是 OSINT 研究者的利器。20k Stars 证明了它在安全社区的认可度。
